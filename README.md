📚 StudyVerse – Online Learning Resources Platform
📌 Project Overview

StudyVerse is a web-based learning resources platform developed using Python and Streamlit. The application provides students with a centralized platform to access various educational materials such as books, videos, audio lectures, notes, and previous question papers.

The system includes user authentication and subscription-based access, allowing users to explore learning resources in a structured and user-friendly interface.

This project aims to support self-learning and digital education by providing categorized learning materials for different subjects, including Computer Science, children's learning content, comics, and academic notes.
